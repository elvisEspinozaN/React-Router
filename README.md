## Overview

A simple React application demonstrating client-side routing using React Router. Features multiple color-themed pages.

## Features

- Client-side routing with React Router DOM
- Four themed routes (Home, Red, Blue, About)

## Technologies Used

- React Router DOM
- Vite
- Netlify
